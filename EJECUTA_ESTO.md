# 🎯 EJECUTA ESTO AHORA

## UN SOLO COMANDO:

```batch
DEPLOY_AUTOMATICO.bat
```

## ¿Qué hace?

1. ✅ Verifica todo automáticamente
2. 📝 Te pide la URL de GitHub (la creas en 1 minuto)
3. 🚀 Sube TODO el código automáticamente
4. 📋 Te da instrucciones para Railway

## Tiempo total: 8 minutos

---

## Si prefieres hacerlo manual:

### 1. Crear repo en GitHub (2 mins)
- Ve a https://github.com/new
- Nombre: `PRISLAB_SaaS`
- Click "Create repository"
- Copia la URL

### 2. Subir código (30 segundos)
```bash
git remote add origin https://github.com/TU_USUARIO/PRISLAB_SaaS.git
git branch -M main
git push -u origin main
```

### 3. Railway (5 mins)
- https://railway.app → Login with GitHub
- New Project → Deploy from GitHub
- Selecciona PRISLAB_SaaS
- Agrega PostgreSQL
- Configura 5 variables
- ¡Listo!
